# 地铁LCD生成器 (Subway LCD Generator)

一个轻量级、完全运行在浏览器客户端的高保真**地铁乘客信息系统 (PIS) LCD 动态线路图生成器**。支持标准格式和 RailMapGen (RMG) 生成文件的双向导入，支持各种线路配置、换乘标识定义、支线选择、自定义 Logo 上传以及超高清图片导出，可用于地铁爱好者沙盘模拟、二次创作或 PIS 方案排版演示。

##  功能特性

### 1.  强大且直观的配置面板
- **全局配置**：轻松自定义地铁公司名称（中/英文）、顶部时间（支持实时同步系统时间）、线路主题色、已驶过与未驶过轨道的颜色等。
- **动态终点站/下一站**：支持根据当前的“当前站/下一站”标记自动倒推与生成下一站，并能根据列车行驶方向自动侦测首尾站设为终点站。支持手动 Override 覆盖文本。
- **自定义 Logo & 品牌**：支持一键开关 Logo 显示。内置上传控件，用户上传任意图片后将以 Base64 格式缓存，并自动裁剪为完美的圆形 Logo 嵌入屏中，点击“清除”可一键恢复默认的文字 Logo。

### 2.  灵活的站点管理与换乘设置
- **站点维护**：支持自由添加、删除站点，或通过微调按钮一键调整站点上下排布顺序。
- **状态切换**：可为每个站点单独配置运营状态：
  - `已驶过 (Passed)`
  - `当前/下一站 (Current)`
  - `未到达 (Future)`
  - `暂缓开通 (Closed)`（站点圆圈打红叉并附带垂直“暂缓开通”标签）
- **高自由度换乘徽标编辑**：
  - 点击“+ 换乘”即可为站点新增换乘。
  - 支持**数字号线（Digit Style）**（方格大字，右侧带“号线/Line X”文字）和**命名线路（Rect Style）**（圆角横幅徽标，上汉字下英文）。
  - **点击编辑**：轻点编辑器中的换乘徽标即可在弹窗内直接编辑其名称、样式与十六进制标志色（提供色彩选择器同步），弹窗内自带“删除”按钮。

### 3.  RMG 兼容导入与支线决策 ("驶入支线")
- **RMG JSON 识别**：勾选“RMG 兼容导入模式”后，可完美识别 RailMapGen 生成的轨道交通配置文件，自动过滤多余的英文坐标后缀，映射北京地铁等换乘编码主题。
- **双路径支线遍历**：当导入的 RMG 文件中包含分叉支线（如 20 号线）时，左侧会自动激活“**驶入支线 (Enter Branch)**”复选框：
  - **未勾选时**：列车沿主线行驶（例如：九渡河 $\rightarrow$ 九渡河西 $\rightarrow$ 岭南）。
  - **勾选后**：列车沿支线行驶（例如：九渡河 $\rightarrow$ 九渡河南 $\rightarrow$ ... $\rightarrow$ 北岭路）。

### 4.  高保真 LCD 跑马灯预览与超清导出
- **屏幕主题切换**：支持一键开启“深色模拟 (Dark LCD Mode)”，适配暗黑屏显美学。
- **分段轨道渲染**：彻底重构了传统线性渐变，改用 discrete 分段 div 连接相邻站点。颜色切换精确发生在站点中心，在高 DPI 或缩放屏下无缝无空隙，完美解决传统渐变造成的文字下方轨道颜色分裂问题。
- **超清图片导出**：集成 `html2canvas` 导出方案，在导出时临时将缩放设为 1:1，并采用 **3.5倍物理像素级超采样** 渲染，一键下载 crisp、无毛边的 1550x460 高清 PNG 原图。

##  技术栈
- **核心逻辑**：纯 JavaScript (ES6) 编写，无任何前端复杂框架依赖，秒级加载。
- **页面布局**：HTML5 + Vanilla CSS3，广泛应用 CSS Grid / Flexbox 弹性布局、CSS 变量、斜切 `clip-path` 梯形美学与毛玻璃 (Glassmorphic) 卡片。
- **渲染库**：`html2canvas.min.js` (集成 CDN 版本，负责 DOM 树 Canvas 二次渲染)。
- **字体族**：支持 `Noto Sans SC` (思源黑体)、`Outfit` 与 `Arial Bold`，数字采用 upright 无斜体 Arial 设计，完美拟真地铁站牌刻度。

##  数据配置说明

本工具支持双向数据导入与导出：
1. **生成器标准格式**：
   - 导出的 `.json` 携带全部生成器配置（如 Logo 的 Base64 URL 字符串、是否实时时间、已/未到颜色值等）以及 `stations` 数组。
2. **RMG 兼容格式**：
   - 勾选 RMG 兼容模式后，直接选中 RMG 导出的原装 JSON 载入即可。
